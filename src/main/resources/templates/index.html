Նկարագրություն

Ինտերնետային ֆորում [այս դեպքում ՝ ծրագրավորման մասին], որը նախագծվել է Spring Framework- ի և MySQL- ի օգտագործմամբ: Այս վեբ հավելվածը հնարավորություն է տալիս, ավելացնել թեմաներ ըստ կատեգորիաների,պատասխանել արդեն իսկ քննարկվող հարցերի, փնտրել հարցեր:

Օգտագործված տեխնոլոգիաներ

Java 11 Standard Edition, Spring MVC, Spring Data, Spring Security, Spring Boot, JPA/Hibernate, HTML (Themyleaf), CSS (Materialize)

Ծրագրային գործիքներ

IntelliJ IDEA, Gradle, Docker
